

## Description du Projet
Ce projet est une application Java qui gère un environnement composé d'un hangar et d'une ferme, permettant de recenser et de gérer divers types de véhicules et d'animaux avec des capacités spécifiques telles que voler, rouler et naviguer.

## Fonctionnalités
- Gestion des véhicules dans un hangar.
- Gestion des animaux dans une ferme.
- Recensement des capacités de mouvement (vol, roulement, navigation).

## Technologies Utilisées
- **Java JDK 17:** Langage de programmation.
- **IDE:** IntelliJ IDEA.

## Structure du Projet
Le projet est organisé en plusieurs packages pour une meilleure modularité :
- `org.nicodeme.contracts`: Contient les interfaces pour les capacités de déplacement.
- `org.nicodeme.environnement`: Contient les classes pour la gestion de l'environnement, du hangar, et de la ferme.
